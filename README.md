# 🏠 homepage的主页

## ✨ 项目功能

- 🎨 **简洁美观** - 提供清爽的主页展示界面
- 📱 **响应式设计** - 完美适配手机、平板、桌面等各种设备
- ⚡ **极速加载** - 优化性能，提升用户浏览体验

## ⚙️ 使用说明

1. **配置 GitHub 统计**
    - 打开 `static/script.js` 文件
    - 修改第一行的 GitHub username 为你的用户名

2. **部署方式**
    - 安装依赖 `npm install`
    - 启动服务器 `node server.js`

3. **后台地址**
    - 访问 http://localhost:3000/admin
